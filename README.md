# Real Estate Market Intelligence Platform
Welcome to the **Real Estate Market Intelligence Platform**, an AI-driven single-page executive dashboard designed to provide deep insights into buyer segmentation and investment profiling.

## Project Overview
This application leverages advanced data analytics and machine learning to analyze client behavior, geographic distribution, and property investment patterns. It segments clients into distinct groups to help make data-driven business decisions for real estate portfolios.

## Key Features
 * **Buyer Segmentation**: Uses K-Means clustering to categorize clients into segments such as Global Investors, First-Time Buyers, Corporate Buyers, and Luxury Investors.
 * **Geographic Buyer Analysis**: Visualizes buyer volume and transnational inbound matrix flows across different regional jurisdictions.
 * **Investor Behavior Dashboard**: Displays strategic intent and acquisition purposes through interactive charts and cross-tabulations.
 * **High-Value Customer Risk Explorer**: Provides dynamic revenue risk profiling and compares financial assets between retained and churned customers.

## Technologies Used
 * **Python**
 * **Streamlit** for building the interactive web application.
 * **Pandas** & **NumPy** for data manipulation and processing.
 * **Scikit-Learn** for machine learning (K-Means clustering, StandardScaler, LabelEncoder).
 * **Plotly Express** & **Seaborn** for data visualization.

## Installation
To run this application locally, ensure you have Python installed, then install the required dependencies:
```bash
pip install streamlit pandas numpy scikit-learn plotly seaborn matplotlib

```
## How to Run
After installing the dependencies, run the application using the following command in your terminal:
```bash
streamlit run RealEstateMarketIntelligencePlatform.py

```
